# 学生课程成绩管理系统 (Student Course Grade Management System)

## 项目简介

本项目为上海大学【2025-2026学年秋季学期《面向对象程序设计A》课程】实验一

本项目是一个基于 C++ `LinkList` 类模板实现的综合应用程序。系统采用**三层结构**设计，核心数据结构为**链表的链表**（Linked List of Linked Lists），用于管理学生及其选修课程的成绩信息。

## 数据结构设计

### 1. 核心类模板
项目依赖于 [`LinkList`](include/LinkList.h) 类模板（需确保 `include/LinkList.h` 存在并正确实现）。

### 2. 数据模型
系统包含两个主要的数据实体：

*   **Course (课程)**: 存储单门课程的信息（如课程名、分数）。
*   **Student (学生)**: 存储学生基本信息（如姓名、学号），并包含一个 `LinkList<Course>` 对象来管理该学生的所有课程。

### 3. 链表结构 (链表的链表)
*   **外层链表**: `LinkList<Student>`
    *   每个节点代表一个学生。
    *   支持添加学生、删除学生、查找学生。
*   **内层链表**: `LinkList<Course>` (位于 `Student` 类内部)
    *   每个节点代表该学生的一门课程成绩。
    *   支持录入成绩、修改成绩、删除课程。

## 功能功能

1.  **学生管理**
    *   注册新学生
    *   注销学生信息
    *   按学号查找学生

2.  **课程与成绩管理**
    *   为指定学生添加课程成绩
    *   修改指定课程的成绩
    *   删除课程记录

3.  **统计与报表**
    *   打印所有学生及其成绩单
    *   计算每个学生的平均分
    *   (可选) 统计某门课程的平均分

## 编译与运行

本项目使用 CMake 构建。

1.  **构建项目**:
    ```bash
    mkdir build
    cd build
    cmake ..
    make
    ```

2.  **运行程序**:
    ```bash
    ./main
    ```
